Hola Mundo


Esta es la primera pagina web del curso, muestra la estructura minima basica de una pagina web

Un archivo html (con extension .html) comienza y termina con la etiqueta html que delimita al documento, luego viene el encabezado, en esa seccion se coloca el titulo de la pagina (dentro de etiquetas title) alli tambien se agregan algunos elementos adicionales de scripts y metas que se veran luego.

Ya luego viene el cuerpo de la pagina, lo que sera visible a los usuarios, visitantes o internautas, esto va dentro de etiquetas body
